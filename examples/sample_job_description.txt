Senior Machine Learning Engineer

About Us:
We are a leading AI company revolutionizing how businesses leverage machine learning. We're looking for a talented ML Engineer to join our growing team.

Job Description:
We are seeking an experienced Machine Learning Engineer to design, develop, and deploy ML models at scale. You will work on cutting-edge projects involving natural language processing, computer vision, and predictive analytics.

Key Responsibilities:
• Design and implement machine learning models using TensorFlow and PyTorch
• Build and maintain ML pipelines for model training and deployment
• Collaborate with data scientists to translate prototypes into production systems
• Optimize model performance and ensure scalability
• Implement A/B testing frameworks for model evaluation
• Work with cloud platforms (AWS, GCP) for model deployment
• Mentor junior team members and conduct code reviews

Required Qualifications:
• 5+ years of experience in machine learning or related field
• Strong proficiency in Python and ML frameworks (TensorFlow, PyTorch, Scikit-learn)
• Experience with deep learning architectures (CNNs, RNNs, Transformers)
• Hands-on experience with cloud platforms (AWS SageMaker, GCP Vertex AI)
• Solid understanding of statistics and mathematics
• Experience with MLOps practices and tools
• Excellent problem-solving and communication skills

Preferred Qualifications:
• Master's or PhD in Computer Science, Machine Learning, or related field
• Experience with NLP libraries (Hugging Face, spaCy)
• Knowledge of distributed computing (Spark, Ray)
• Familiarity with containerization (Docker, Kubernetes)
• Published research papers or open source contributions
